# Bookmark

### 

## Javascript Code
```js
window.dataLayer = window.dataLayer || [];
dataLayer.push({
  "event": "Bookmark",
  "event_data": {
    "location": "<location>"
  }
});
```


## Variable Definitions

|Path|Type|Description|
| --- | --- | --- |
|event_data.location|string|Captures the location where page is bookmarked.|
