# Page Browser Section

### 

## Javascript Code
```js
window.dataLayer = window.dataLayer || [];
dataLayer.push({
  "event": "PageBrowserSection",
  "event_data": {
    "itemName": "<itemName>"
  }
});
```


## Variable Definitions

|Path|Type|Description|
| --- | --- | --- |
|event_data.itemName|string|Captures the clicked section.|
