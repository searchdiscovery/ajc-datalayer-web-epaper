# Search

### 

## Javascript Code
```js
window.dataLayer = window.dataLayer || [];
dataLayer.push({
  "event": "search",
  "event_data": {
    "search_term": "<search_term>"
  }
});
```


## Variable Definitions

|Path|Type|Description|
| --- | --- | --- |
|event_data.search_term|string|Captures the term that was searched.|
